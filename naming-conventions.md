## General

* all lowercase kabob-case
* ex: hello-world

## Files and Object names

* filename is `metadata.name` and resource type.
* ex:
  - kind: Service
  - metadata.name: frontend 
  - frontend-service.yaml

## Labels

Use semantic attributes:

```yaml
app: webapp
tier: frontend # or front-end
env: test
version: v3
```
